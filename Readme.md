# Simple estimate file space usage

A simple tool for size analyze files in your file system like `du` command.

# Setup

Clone this repository and enter to new folder:

```
git clone https://github.com/sr2ds/du-rs && cd du-rs
```

Run with Cargo:
```
cargo watch -x run
```
